* pwd
* ls
* cd
* mkdir
* rm

### File Viewing

* cat
* less
* head
* tail

### Permissions

* chmod
* chown

### Networking

* ping
* nslookup
* traceroute

To build a strong foundation in Linux for cybersecurity, networking, and system administration.
